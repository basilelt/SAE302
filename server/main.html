Main Module¶

main.main()¶

Main function to start the chat server.

This function parses command line arguments for host and port, validates them, and starts the server.

Raises:

  • KeyboardInterrupt – If the server is interrupted by keyboard input

  • Exception – If there is an error starting the server

SAE302-Server
